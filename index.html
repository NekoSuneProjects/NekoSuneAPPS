🎬 TikTok

          

            Followers without being live: type the creator's @username and press Track followers. Reads the public profile, refreshes every 2 min.

            Live (viewers + new follows): press Connect live — the creator must be streaming.

            SignatureError / 404? TikTok requests are signed via Euler Stream. Get a free API key there and paste it in the TikTok card.

            "Unexpected server response: 200" = the live event socket couldn't open (free signing tier). Follower tracking still works.
          

        


        

          
🔗 OAuth redirect URLs (one shared port)

          

            NekoSuneAPPS uses a single local port (3737) for all logins, namespaced by service. Register these in each developer portal:

            • Twitch: http://localhost:3737/oauth2/twitch/callback

            • Discord (voice only): http://localhost:3737/oauth2/discord/callback

            Nothing is exposed to the internet — these are loopback URLs the app listens on only during login.
          

        


        

          
🟣 Twitch (OAuth2)

          

            1. Create an app at dev.twitch.tv/console and copy its Client ID.

            2. Add the Twitch redirect URL above to the app's OAuth Redirect URLs.

            3. Copy the Client ID into the Twitch card. Client Secret is optional — leave it blank for a quick login (token expires in ~4h), or add it to get auto-refreshing tokens.

            4. Press 🔑 Login with Twitch, approve, done.

            5. Enter your channel login and press Start.
          

        


        

          
🟢 Kick

          

            Enter your channel slug (the part after kick.com/) and press Connect. No login needed.
            If it shows 403 (Cloudflare), Kick is blocking the request — a browser fallback can be enabled (ask to wire it).
          

        


        

          
❤️ Pulsoid heart rate

          

            Get a token at pulsoid.net/ui/keys with scope data:heart_rate:read, paste it in Stats → Heart rate, press Start. The line shows bpm | avg | max | min.
          

        


        

          
🤖 IntelliChat AI

          

            Pick a Provider in Settings. Any OpenAI-compatible gateway works:

            • OpenAI — key from platform.openai.com

            • Ollama (local) — run ollama serve, base http://localhost:11434/v1, no key

            • LiteLLM — point base URL at your gateway (e.g. http://localhost:4000/v1)

            • xAI (Grok), Google Gemini, Groq, OpenRouter, Anthropic — pick the preset and paste that provider's key

            • Custom — any other compatible endpoint.
          

        


        

          
💜 Discord

          

            Rich Presence (show VRChat status on your profile): create an app at
            discord.com/developers,
            paste its Application (Client) ID into the Discord card, and press Connect —
            no secret, no redirect needed.


            Voice detection (channel name, speaking, mute/deafen) needs Discord's private
            rpc.voice.read scope, which Discord only grants to allowlisted apps.
            Unapproved apps get invalid_scope and the app falls back to Rich Presence.
            To enable it you must apply for RPC access for your Discord application at
            Discord's RPC docs,
            then add a Client Secret + the Discord redirect URL above. Once approved, the login is
            remembered (no re-auth each launch).
          

        


        

          
🖥 OBS overlay

          

            Enable it in Integrations, then add the shown URL as a Browser Source in OBS.
          

        


        

          
🔋 VR gear battery — does it work without VR?

          

            No. VR battery reads your SteamVR / OpenVR devices (headset, controllers, trackers). With no VR headset and SteamVR not running there are no devices to report, so it stays "unavailable". It is only useful while you're in VR with SteamVR running — desktop-only users can leave it off.
